 ### Bash Group Exercise

Use the data in this [link](https://github.com/kipkurui/Intro2Linux2019/raw/master/Data/protein.fa.gz) for this exercise.

1. Create a new project, complete with all the directories you would need for a reproducible project. Which commands did you use?
2. Using the command line, download and uncompress the data into the appropriat directory
3. From the file `protein.fa`, create a new FASTA file which only contains uncharacterized proteins, and a clean FASTA file without uncharacterized sequences
4. Create a new file which only contains raw sequences without the identifying information, or new lines. How many amino acids are in this new file?
